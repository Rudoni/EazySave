EazySave est une application permettant de créer la sauvegarde d'éléments de manière totale ou différentielle.

Téléchargement de l'application :
https://github.com/Rudoni/EazySave

----------------------------------------------------------------

Guide d'utilisation :

1. Création d'une sauvegarde

Créer une sauvegarde -> Entrer le nom de la sauvegarde, le chemin source et le chemin de destination -> Choisir entre sauvegarde totale et différentielle

2. Lancement d'une sauvegarde

Charger une sauvegarde -> Entrer le numéro d'une ou plusieurs sauvegardes ->  La sauvegarde est réalisée

3. Paramètres de de l'application

Paramètres -> Choix de la langue -> Choix entre les langues disponibles
